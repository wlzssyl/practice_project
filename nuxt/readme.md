#### 1.全局安装npx
>npm install -g npx
#### 2.创建项目
>npx create-nuxt-app project-name
#### 3.项目目录
  - assets 项目资源文件
  - components 组件
  - layouts 模板
  - middleware 中间件
  - pages 页面
  - plugins 插件
  - server 服务端
#### 4.辅助工具安装
  - sass安装
    >npm i sass-loader node-sass
  - esint安装(可能不需要)
    >npm i eslint 
  - nuxt.config.js配置
    > //Global CSS
    >'element-ui/lib/theme-chalk/reset.css'
